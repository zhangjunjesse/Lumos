# Lumos 任务执行架构

## 架构分层

```
┌──────────────────────────────────────────────────────┐
│                    用户层                             │
│                 (User Layer)                         │
│  • 对话界面  • 任务查询  • 结果展示                   │
└──────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────┐
│                  Main Agent                          │
│              (对话入口 & 决策者)                       │
│  • 意图识别  • 任务评估  • 下发决策                   │
│  • 结果呈现  • 上下文维持                             │
└──────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────┐
│            Task Management Layer                     │
│              (任务状态管理中心)                        │
│  • 任务注册  • 状态跟踪  • 优先级管理                 │
│  • 快速查询  • 任务取消/暂停                          │
└──────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────┐
│              Scheduling Layer                        │
│            (任务执行管理 - 项目经理)                   │
│  • 任务拆解  • 角色判断  • 输出物定义                 │
│  • 执行规划  • 生命周期管理  • 进度跟踪               │
└──────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────┐
│               SubAgent Layer                         │
│              (资源层 - 人力资源部)                     │
│  • 角色创建  • 能力匹配  • 资源分配                   │
│  • Agent外派  • 生命周期管理                          │
└──────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────┐
│                 结果存储层                            │
│               (Storage Layer)                        │
│  • 任务历史  • 执行日志  • 产物存储                   │
│  • 状态持久化  • 查询索引                             │
└──────────────────────────────────────────────────────┘
```
