## 一、设计系统

### 1.1 颜色方案

#### 主色调
- **Primary Blue**: `#3B82F6` - 主要按钮、链接
- **Primary Blue Hover**: `#2563EB` - 主要按钮悬停
- **Primary Blue Active**: `#1D4ED8` - 主要按钮点击

#### 状态颜色
- **Success Green**: `#10B981` - 成功状态、已绑定徽章
- **Warning Yellow**: `#F59E0B` - 警告状态、已暂停徽章
- **Error Red**: `#EF4444` - 错误状态、危险按钮
- **Error Red Hover**: `#DC2626` - 危险按钮悬停
- **Error Red Active**: `#B91C1C` - 危险按钮点击
- **Info Blue**: `#3B82F6` - 信息提示

#### 中性色
- **Gray 50**: `#F9FAFB` - 浅背景
- **Gray 100**: `#F3F4F6` - 输入框背景、悬停背景
- **Gray 200**: `#E5E7EB` - 边框
- **Gray 300**: `#D1D5DB` - 禁用状态、分隔线
- **Gray 400**: `#9CA3AF` - 占位符文字
- **Gray 500**: `#6B7280` - 次要文字、说明文字
- **Gray 600**: `#4B5563` - 次要按钮悬停
- **Gray 700**: `#374151` - 次要按钮点击
- **Gray 900**: `#111827` - 主要文字

#### 背景色
- **White**: `#FFFFFF` - 主背景、卡片背景
- **Black**: `#000000` - 遮罩层（透明度 50%）

### 1.2 字体系统

#### 字体家族
- **Sans-serif**: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`
- **Monospace**: `"SF Mono", Monaco, "Cascadia Code", "Roboto Mono", Consolas, monospace`

#### 字体大小
- **Heading 1**: `24px` / `1.5rem` - 对话框标题
- **Heading 2**: `18px` / `1.125rem` - 面板标题
- **Heading 3**: `16px` / `1rem` - 群组名称、统计项数字
- **Body**: `14px` / `0.875rem` - 正文、按钮文字
- **Caption**: `12px` / `0.75rem` - 说明文字、时间戳

#### 字重
- **Regular**: `400` - 正文
- **Medium**: `500` - 按钮、标签
- **Semibold**: `600` - 群组名称、统计项数字
- **Bold**: `700` - 标题

#### 行高
- **Tight**: `1.25` - 标题
- **Normal**: `1.5` - 正文
- **Relaxed**: `1.75` - 说明文字

### 1.3 间距系统

#### 基础间距单位
- **Base**: `4px` - 最小间距单位

#### 间距规范
- **xs**: `4px` - 图标与文字间距
- **sm**: `8px` - 统计项间距、按钮内边距（垂直）
- **md**: `12px` - 按钮内边距（水平）、卡片内边距
- **lg**: `16px` - 面板内边距、对话框内边距
- **xl**: `24px` - 对话框内边距（顶部/底部）
- **2xl**: `32px` - 大间距

### 1.4 圆角系统

- **None**: `0px` - 无圆角
- **sm**: `4px` - Toast、小按钮
- **md**: `6px` - 按钮、输入框
- **lg**: `8px` - 卡片、面板
- **xl**: `12px` - 开关
- **full**: `9999px` - 徽章、圆形按钮

### 1.5 阴影系统

- **sm**: `0 1px 2px rgba(0, 0, 0, 0.05)` - 轻微阴影
- **md**: `0 4px 6px rgba(0, 0, 0, 0.1)` - 中等阴影
- **lg**: `0 10px 15px rgba(0, 0, 0, 0.1)` - 较大阴影
- **xl**: `0 20px 25px rgba(0, 0, 0, 0.15)` - 对话框阴影
- **popover**: `0 4px 12px rgba(0, 0, 0, 0.15)` - Popover 阴影

### 1.6 动画系统

#### 过渡时间
- **Fast**: `150ms` - 按钮状态切换
- **Normal**: `200ms` - 面板进入/退出
- **Slow**: `300ms` - 对话框进入/退出

#### 缓动函数
- **Ease Out**: `cubic-bezier(0, 0, 0.2, 1)` - 进入动画
- **Ease In**: `cubic-bezier(0.4, 0, 1, 1)` - 退出动画
- **Ease In Out**: `cubic-bezier(0.4, 0, 0.2, 1)` - 状态切换

## 五、错误提示设计（Toast）

### 5.1 Toast 通用样式

**位置**：屏幕右上角，距离顶部 `16px`，距离右侧 `16px`

**样式规范**：
- 宽度：`360px`
- 高度：自适应
- 圆角：`4px`
- 阴影：`0 4px 12px rgba(0, 0, 0, 0.15)`
- 内边距：`12px 16px`

**布局结构**：

```
┌─────────────────────────────────────────┐
│  [图标] [文字内容]              [关闭] │
│         [操作按钮]（可选）              │
└─────────────────────────────────────────┘
```

**动画**：
- 进入：从右侧滑入 + 淡入，`300ms ease-out`
- 退出：向右侧滑出 + 淡出，`200ms ease-in`
- 自动关闭：持续时间后自动退出

### 5.2 成功提示（Success Toast）

- 其他：3 秒

### 5.3 错误提示（Error Toast）

## 七、可访问性设计

### 7.1 颜色对比度

**WCAG AA 标准**（对比度 ≥ 4.5:1）：

- 主要文字（`#111827`）vs 白色背景（`#FFFFFF`）：✅ 16.1:1
- 次要文字（`#6B7280`）vs 白色背景（`#FFFFFF`）：✅ 4.6:1
- 主要按钮文字（`#FFFFFF`）vs 蓝色背景（`#3B82F6`）：✅ 4.5:1
- 危险按钮文字（`#FFFFFF`）vs 红色背景（`#EF4444`）：✅ 4.5:1
- 成功徽章文字（`#FFFFFF`）vs 绿色背景（`#10B981`）：✅ 4.5:1

### 7.2 键盘导航

**Tab 顺序**：
1. 按钮（"同步到飞书"、"复制链接"、"解绑"等）
2. 输入框（分享链接输入框）
3. 开关（同步开关）
4. 关闭按钮（对话框、面板）

**快捷键**：
- `Esc`：关闭对话框、面板
- `Enter`：确认操作（对话框中）
- `Space`：切换开关

**焦点样式**：
- 边框：`2px solid #3B82F6`
- 偏移：`2px`
- 圆角：继承元素圆角

### 7.3 屏幕阅读器

**ARIA 属性**：

**按钮**：
- `role="button"`
- `aria-label="同步到飞书"`
- `aria-busy="true"`（加载时）
- `aria-disabled="true"`（禁用时）

**状态徽章**：
- `role="button"`
- `aria-label="已绑定，点击查看详情"`
- `aria-haspopup="true"`

**开关**：
- `role="switch"`
- `aria-checked="true"`（ON）或 `"false"`（OFF）
- `aria-label="同步开关"`

**对话框**：
- `role="dialog"`
- `aria-modal="true"`
- `aria-labelledby="dialog-title"`

**Toast**：
- `role="alert"`
- `aria-live="polite"`（信息提示）或 `"assertive"`（错误提示）

### 7.4 动画减弱

**用户偏好**：`prefers-reduced-motion: reduce`

**处理方式**：
- 禁用所有过渡动画
- 禁用缩放、滑动动画
- 保留淡入/淡出（持续时间减少到 100ms）
