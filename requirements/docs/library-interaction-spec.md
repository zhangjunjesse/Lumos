- **点击**：切换筛选类型

## 🃏 卡片网格（Card Grid）

### 网格布局
- **桌面**：grid grid-cols-3 gap-6
- **平板**：grid grid-cols-2 gap-4
- **移动**：grid grid-cols-1 gap-4
- **容器**：max-w-6xl mx-auto px-8

### 卡片设计

#### 卡片结构
```
┌─────────────────────────────────────┐
│ [Logo]  标题                    [⭐] │
│                                     │
│ 预览文本...                          │
│                                     │

│                                     │
│ 📄 文档 · 2024-01-15 · 3 分钟       │

```

#### 尺寸
- **宽度**：flex-1（自适应）
- **高度**：固定高度（避免布局抖动）
  - 标题区：h-12
  - 预览区：h-20（3 行文本）
  - 标签区：h-8
  - 元信息区：h-6
  - 总高度：约 180px

#### 样式
- **背景**：bg-card
- **边框**：border border-border
- **圆角**：rounded-lg
- **阴影**：shadow-sm
- **内边距**：p-4

#### 状态

1. **默认状态**
   - 边框：border-border
   - 阴影：shadow-sm

2. **Hover 状态**
   - 边框：border-primary/50
   - 阴影：shadow-md
   - 变换：scale-[1.02]
   - 动画：transition-all duration-200

3. **选中状态**（多选模式）
   - 边框：border-primary border-2
   - 背景：bg-primary/5
   - 复选框：显示并选中

4. **加载状态**
   - 显示骨架屏
   - 动画：animate-pulse

#### 组件详情

##### 1. Logo 区域
- **尺寸**：w-10 h-10
- **圆角**：rounded
- **背景**：根据类型显示不同颜色
  - 文档：bg-blue-100
  - 对话：bg-green-100
  - 音频：bg-purple-100
  - 视频：bg-red-100
  - 网页：bg-yellow-100
- **图标**：居中显示类型图标

##### 2. 标题
- **字体**：text-base font-semibold
- **颜色**：text-foreground
- **行数**：line-clamp-2（最多 2 行）
- **Hover**：text-primary（鼠标悬停时）

##### 3. 收藏按钮
- **位置**：右上角
- **图标**：⭐（已收藏）/ ☆（未收藏）
- **尺寸**：w-5 h-5
- **交互**：
  - 点击：切换收藏状态

##### 4. 预览文本
- **字体**：text-sm
- **颜色**：text-muted-foreground
- **行数**：line-clamp-3（最多 3 行）
- **行高**：leading-relaxed

##### 5. 标签区域
- **布局**：flex gap-2 flex-wrap
- **最多显示**：3 个标签
- **超出显示**：+2（显示剩余数量）

**标签样式**：
- 背景：bg-secondary
- 文字：text-secondary-foreground text-xs
- 圆角：rounded-full
- 内边距：px-2 py-1

##### 6. 元信息
- **布局**：flex items-center gap-2
- **字体**：text-xs
- **颜色**：text-muted-foreground
- **分隔符**：·

**显示内容**：

- **触发**：

## 📊 性能优化

### 虚拟滚动
- 只渲染可见区域的卡片
- 使用 `react-window`
- 预加载上下 5 个卡片

### 图片懒加载
- 使用 Intersection Observer
- 占位符：模糊图片或颜色块
- 渐进式加载

### 代码分割
- 路由级别分割
- 组件级别分割（详情页、编辑器等）
- 动态导入

### 缓存策略
- API 响应缓存（5 分钟）
- 图片缓存（永久）
- 搜索结果缓存（1 分钟）
