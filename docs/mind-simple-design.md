# Lumos Mind 模块 - 极简设计方案

## 🎯 设计原则

**极简、优雅、自然语言**

- 只展示用户需要的信息
- 隐藏所有技术细节
- 用自然语言而非技术术语

---

## 📐 界面结构

### 只保留 2 个 Tab

```
┌─────────────────────────────────────┐
│  Lumos 的心智                       │
│  [概览] [编辑]                      │
└─────────────────────────────────────┘
```

**概览 Tab**（默认）：
- 一眼看懂 Lumos 的"心智"
- 自然语言展示
- 无技术统计

**编辑 Tab**：
- 需要修改时才进入
- 表单式编辑
- 实时预览

---

## 🎨 概览页面设计

### 布局


### 视觉特点

- 大圆角（20px）
- 柔和渐变背景
- 大量留白（padding: 32px）
- 轻微阴影（0 2px 8px）

## ❌ 删除的内容

**不再展示**：
- hitCount（使用次数统计）
- lastUsedAt（最后使用时间）
- category（技术分类）
- scope（作用域）
- projectPath（项目路径）
- 8个Tab（统计、配置、历史等）
- 归档、删除按钮（移到右键菜单）

---

## ✅ 保留的内容

**只展示核心**：
- 用户画像（自然语言）
- AI 性格（一句话概括）
- 相处方式（一句话概括）
