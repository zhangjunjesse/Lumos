# Lumos 用户体验设计 v2.0
## 核心理念：做最懂主人的 agent

---

## 一、产品定位重新定义

### 从"培养 AI"到"让 AI 了解你"

**旧思路 ❌：**
- 用户配置 AI 的性格和能力
- 像"训练宠物"一样塑造 AI
- 重点在 AI 如何表现

**新思路 ✅：**
- AI 主动了解主人
- 像"贴心管家"一样理解主人
- 重点在 AI 如何理解你

---

## 二、三个模块重新定位

### 1. 主人画像（核心模块）⭐

**Lumos 眼中的你**

```
┌─────────────────────────────────────┐
│ 👤 主人画像                          │
│                                     │
│ Lumos 对你的了解程度：68%           │
│ ████████████░░░░░░░░                │
│                                     │
│ 基本信息 ✓                          │
│ 工作背景 ✓                          │
│ 沟通偏好 ⚠️ 需要更多互动             │
│ 专业领域 ✓                          │
│ 思维方式 ⚠️ 持续学习中               │
│                                     │
│ [查看详细画像] [帮助 Lumos 更懂你]   │
└─────────────────────────────────────┘
```

**核心内容：**
- 基本信息（称呼、职业、背景）
- 工作方式（习惯、偏好、节奏）
- 思维特点（逻辑型/直觉型、细节/全局）
- 沟通风格（简洁/详细、正式/随意）
- 专业领域（技术栈、行业知识）
- 当前状态（项目、目标、挑战）

### 2. Lumos 的人格

**AI 如何表现**

- 性格基调（专业/友善/活泼）
- 回答风格（简洁/详细）
- 语言特点（正式/轻松）

### 3. 相处规则

**如何互动**

- 什么时候主动提醒
- 如何处理不确定的问题
- 边界和原则

## 七、数据结构调整

### 主人画像数据结构

```typescript
interface MasterProfile {
  // 基本信息
  basic: {
    name: string;
    occupation: string;
    currentProject?: string;
    challenges?: string[];
  };

  // 工作方式
  workStyle: {
    pace: 'fast-iteration' | 'plan-first' | 'flexible';
    problemSolving: 'research-self' | 'ask-others' | 'trial-error';
    detailLevel: 'big-picture' | 'detail-oriented' | 'balanced';
    confidence: number; // 0-100，了解程度
  };

  // 沟通偏好
  communication: {
    answerStyle: 'concise' | 'detailed' | 'adaptive';
    proactiveness: 'reactive' | 'suggestive' | 'proactive';
    formality: 'formal' | 'casual' | 'balanced';
    confidence: number;
  };

  // 专业领域
  expertise: {
    domains: string[];
    skills: Array<{
      name: string;
      level: number; // 0-100
      lastUpdated: Date;
    }>;
    confidence: number;
  };

  // 思维特点
  thinking: {
    style: 'logical' | 'intuitive' | 'mixed';
    decisionMaking: 'data-driven' | 'experience-based' | 'balanced';
    confidence: number;
  };

  // 学习记录
  learningLog: Array<{
    date: Date;
    observation: string;
    category: 'basic' | 'workStyle' | 'communication' | 'expertise' | 'thinking';
  }>;

  // 整体了解度
  overallConfidence: number; // 0-100
}
```

## 九、关键差异对比

| 维度 | 旧方案（培养 AI） | 新方案（AI 了解你） |
|------|-----------------|-------------------|
| 核心模块 | SOUL（AI 性格） | 主人画像 |
| 用户角色 | 训练师 | 被了解者 |
| 首次引导 | 配置 AI 的性格和能力 | AI 询问关于你的问题 |
| 持续交互 | 用户调整 AI 设置 | AI 主动学习和询问 |
| 进度展示 | 配置完成度 | 了解程度 |
| 情感连接 | "我的 AI" | "懂我的 AI" |
